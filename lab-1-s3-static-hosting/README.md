# AWS Static Website Hosting Labs ☁️

## 📋 Overview
This repository contains comprehensive documentation and code for AWS cloud computing labs demonstrating static website hosting solutions using Amazon Web Services.

## 🚀 Lab Projects

### [Lab 1: S3 Static Website Hosting](./lab-1-s3-static-hosting)
Basic static website hosting using Amazon S3 bucket with public access configuration.

### [Lab 2: AWS Amplify Deployment](./lab-2-amplify-deployment)
Enhanced deployment using AWS Amplify with GitHub integration and CI/CD capabilities.

## 🛠️ Technologies Used
- Amazon Web Services (AWS)
  - S3 (Simple Storage Service)
  - AWS Amplify
- GitHub (Version Control & CI/CD)
- HTML5/CSS3
- Markdown
